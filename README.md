Контейнер настроен на предварительную компиляцию java приложения. \
Это не затрагивает postgres. \
Настройки подключения к БД задаются в `docker-compose.yml` \
Тестировал только на Linux \
Запуск:
`docker compose up --build`

Добавил в код немного отсебятины вроде работы с конфигами в качестве заготовки на будущее. Думаю этот проект использую потом как заготовку для чего-то другого.
